# UrbanMart_mini_project
UrbanMart is a mini e-commerce web app built with Spring Boot and React. It supports product management, user registration, authentication with JWT, and secure CRUD operations. The project demonstrates full-stack development, REST APIs, validation, and smooth client-server communication.
